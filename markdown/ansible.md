## `clouds.yml`
```yaml
clouds:
  testing:
    auth:
      auth_url: http://openstack.local:5000/v2.0
      username: admin
      password: secret
      project_name: admin
```


## Uploading an image
```yaml
- os_image:
    cloud: testing
    name: cirros
    state: present
    disk_format: qcow2
    container_format: bare
    filename: cirros.qcow2
  register: myimage
```


## Creating a VM
```yaml
- name: create a nova server
  os_server:
    cloud: testing
    name: myserver
    state: present
    nics:
      - net-name: private
    image: "{{ myimage.image.id }}"
    flavor: m1.small
    key_name: my_ssh_key
```
